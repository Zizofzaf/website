⚙️ Code Generator System

This project is a simple automatic code generator system designed for ticket generation, testing, and internal tool demos. It enables users to create unique IDs or codes quickly through a user-friendly interface.

🧠 Key Features

  • Generate unique or random codes (Ticket ID / UUID / Custom)
  • Duplicate check
  • Logging & record keeping
  • Copy-to-clipboard

🛠️ Technologies Used

  • Python
  • Tkinter (Graphical Interface)
  • uuid / random module

🎯 System Advantages

The Ticket ID Generator System ensures the uniqueness of each incident ID by checking against existing entries before generating a new one. This prevents duplication and minimizes human error during ticket creation. With a centralized file acting as the single source of truth, the system allows for efficient management and easy tracking of all incident records.
